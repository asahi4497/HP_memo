Vscode のterminal変更方法

Vscode のterminalをgit-bashに変更する

  • VScode でctrl + P から　setting.json を検索
  • 以下を記述
  • "terminal.integrated.shell.windows":"C:\\Program Files\\Git\\bin\\bash.exe",
    "terminal.integrated.shellArgs.windows":["-l"],

Windows の環境setup

複数のterminal を切り替えられるようにする

参考サイト

git-bashとanacondapromptをvscodeで切り替えて使えるようにしたかった

  • 拡張機能[Shell launcher]を入れる　(デフォルトで2つのterminalは切り替えられない)
  • File -> Preferences -> Setting のsetting.json入るに追記する 
  • 
			"shellLauncher.shells.windows": [
			{
				"shell": "C:\\Windows\\sysnative\\cmd.exe",
				"label": "cmd"
			},
			{
				"shell": "C:\\Windows\\sysnative\\WindowsPowerShell\\v1.0\\powershell.exe",
				"label": "PowerShell"
			},
			{
				"shell": "C:\\Program Files\\Git\\bin\\bash.exe",
				"label": "Git bash"
			},
			{
				"shell": "C:\\Windows\\sysnative\\bash.exe",
				"label": "WSL Bash"
			}
  • 設定を変えたら ctrl + shift + p のコマンドパレットで　「Shell launcher」で検索、選択するとshell の候補が出てくる
  • keyboardsetting でショートカットキーを指定すると楽 (ctrl + shift + 1 に設定した)
